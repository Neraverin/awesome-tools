# awesome-tools
Набор полезных инструментов для администрирования и программирования (мой персональный список)

# Администрирование
## Windows

- [winget](https://github.com/microsoft/winget-cli) - пакетный менеджер для Windows (есть по-умолчанию с Windows 10)
- [terminal](https://github.com/microsoft/terminal) - базовый терминал (есть в winget)
- [MobaXterm](https://mobaxterm.mobatek.net/download.html) - продвинутый терминал, с сессиями, просмотром каталогов, мониторингом и тд. Хорошо подходит для удаленной работы с серверами (есть в winget) (НЕ ПРОВЕРЕНО)

## Linux

- [ghostty](https://github.com/ghostty-org/ghostty) - еще один terminal для Linux (НЕ ПРОВЕРЕНО)

# Разработка
## DevOps
- [unregistry](https://github.com/psviderski/unregistry) - push докер образа напрямую на сервер, без использования Docker registry

## Общее
- [slim](https://github.com/slimtoolkit/slim) - утилита для уменьшения размера Docker образа. Альтернатива - использование образов без ОС вообще
- [mise](https://github.com/jdx/mise) - утилита для установки и работы с несколькими разными версиями dev-фреймворков на рабочей станции. Поволяет установить, например, несколько версий NodeJS, Python и тд. и переключаться между ними
